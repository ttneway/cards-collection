### P2-7 教師班級任務分析與報表中心
- 狀態：待處理
- 優先級：P2
- 問題描述：教師目前缺少可以快速查看班級任務執行情況、學生領點趨勢與個人表現的統整頁面。
- 預期行為：提供教師可直接查看的分析中心，至少包含時間範圍篩選、班級切換、個人報表、任務成效排行與點數趨勢。
- 影響範圍：教師後台、任務紀錄查詢、圖表元件、彙總 RPC 或 view。
- 備註 / 待確認：建議功能拆成幾塊：
  - 班級總覽：總發點數、完成任務次數、活躍學生數、最近 7 天 / 30 天趨勢
  - 任務分析：任務完成次數排行、任務總發點排行、不同時段的完成熱度、任務開啟後的領取轉換
  - 學生報表：個人累積點數、個人完成任務數、常完成任務類型、與班級平均比較、個人時間區間明細
  - 匯出功能：班級報表 CSV、學生明細 CSV、任務統計 CSV

### P3-3 QR Code 辨識穩定性
- 狀態：待處理
- 優先級：P3
- 問題描述：目前不同掃描器對 QR Code 的辨識成功率不一致，仍需整理哪些內容格式與尺寸最穩定。
- 預期行為：確認最穩定的 QR Code 內容格式、尺寸、留白與輸出策略，降低實體掃描失敗率。
- 影響範圍：條碼產生元件、列印版面、下載圖片、掃描器實機測試。
- 備註 / 待確認：需配合實際掃描器做測試紀錄，釐清是內容格式、圖像密度、印刷尺寸還是掃描器相容性問題。

### P3-10 AI 生成卡牌圖片
- 狀態：待處理
- 優先級：P3
- 問題描述：教師目前無法直接用提示詞產生卡牌圖片，也缺少與收集冊主題一致的固定風格流程。

- 影響範圍：教師卡牌管理、圖片生成流程、Supabase Storage、卡牌資料結構。
- 備註 / 待確認：需先定義每本收集冊的視覺風格、提示詞模板與生成後審核流程。
